Modules

transaction_scripts

Rust representation of a Move transaction script that can be executed on the Libra blockchain. Libra does not allow arbitrary transaction scripts; only scripts whose hashes are present in the on-chain script whitelist. The genesis whitelist is derived from this file, and the Stdlib script enum will be modified to reflect changes in the on-chain whitelist as time goes on.
