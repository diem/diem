pub struct OptionFuture<F> { /* fields omitted */ }

Effectively a size=1 future queue, or a slot with at-most-one future inside. This struct is mostly useful for modeling an optional path in futures::select!.

For now, we require F: Unpin which mandates boxing of async fn futures but simplifies the OptionFuture implementation.
