Crate diem_global_constants[−][src]

The purpose of this crate is to offer a single source of truth for the definitions of shared constants within the Diem codebase. This is useful because many different components within Diem often require access to global constant definitions (e.g., Diem Safety Rules, the Key Manager, and Secure Storage). To avoid duplicating these definitions across crates (and better allow these constants to be updated in a single location), we define them here.
