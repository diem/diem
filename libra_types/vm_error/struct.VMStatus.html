pub struct VMStatus {
    pub major_status: StatusCode,
    pub sub_status: Option<u64>,
    pub message: Option<String>,
}

A VMStatus is represented as a required major status that is semantic coupled with with an optional sub status and message.

Fields

major_status: StatusCode

The major status, e.g. ABORTED, OUT_OF_GAS, etc.

sub_status: Option<u64>

The optional sub status. Used e.g. for things such as the abort code, or the arithmetic error type for an ARITHMETIC_ERROR major status.

message: Option<String>

The optional message. Useful for verification errors, and for returning information in validation.
