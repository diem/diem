pub struct NoiseStream<TSocket> { /* fields omitted */ }

A Noise stream with a remote peer.

Encrypts data to be written to and decrypts data that is read from the underlying socket using the noise protocol. This is done by prefixing noise payloads with a u16 (big endian) length field.
