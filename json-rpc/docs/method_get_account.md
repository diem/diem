## Method get_account

**Description**

Get the latest account information for a given account address.


### Parameters

| Name    | Type   | Description                 |
|---------|--------|-----------------------------|
| account | string | Hex-encoded account address |


### Returns

[Account](type_account.md) - If account exists

Null - If account does not exist
